# Docist

A Clojure/script document generator with source and search.

## Usage

### Local

```
$ git clone https://github.com/chadangelelli/docist.git
$ cd docist
$ ./install
```

Docist is in Alpha and therefore can only be used from the REPL at the moment.

```clojure
(require '[docist.core :as dc])
(dc/test!)
```

> View docist.core/test! for current arguments. All this is likely to change once out of Alpha...
